La hipóteis experimental de este estudio es comprobar si el tipo de residuo agroindustrial influye significativamente en la acumulación de PHA en cultivos de Cupriavius necator.

Para contrastar que las diferencias que hay según la fuente de carbono son estadísticamente significativas se puede realizar un test de ANOVA. Además, añadir que la hipótesis planteada es razonable.

3. Formular el modelo lineal necesario para abordar el problema anterior, detallando todos los parámetros que intervienen y las suposiciones de la modelización.

Para este estudio vamos a utilizar el siguiente modelo lineal:

\(y_{ij}\) = \(\mu\) + \(\alpha_{i}\) + \(\epsilon_{ij}\) con la restricción \(\sum_{i=1}^{4} \alpha_i = 0\).

Donde:

  • \(y_{ij}\) es la variable respuesta, es decir, el porcentaje de PHA acumulado respecto al peso seco.

  • \(\mu\) es la media global de la acumulación de PHA.

  • \(\alpha_{i}\) es el efecto del nivel i, es decir, el efecto que tiene cada condición.

  • \(\epsilon_{ij}\) es el error experimental o residuo.

Para que este modelo sea valido, los residuos deben cumplir las siguientes suposiciones de modelización:

  • Las observaciones son independientes entre sí.

  • La varianza de errores es constante en todos los niveles.

  • \(\epsilon_{ij}\) sigue una distribución normal \(\epsilon_{ij} \sim N(0, \sigma^2)\).

En cuanto a los tests de Shapiro-Wilk y Levene se confirma que hay normalidad y homocedasticidad ya que el p-valor es mayor a 0.05.

5. Especificar la hipótesis estadística que se contrasta mediante el ANOVA. Expresarla primero en lenguaje natural y posteriormente en términos paramétricos. Presentar la tabla ANOVA e indicar qué conclusiones se obtienen.

Para determinar si la fuente de carbono influye significativamente en la acumulación de PHA, contrastamos las siguientes hipótesis:

  • Hipótesis nula (\(H_0\)) si no existen diferencias significativas en el porcentaje medio de PHA acumulado entre las diferentes fuentes de carbono. En términos paramétricos: \(H_0: \alpha_1 = \alpha_2 = \alpha_3 = \alpha_4 = 0\).

  • Hipótesis alternativa (\(H_1\)) si al menos una de las fuentes de carbono produce un porcentaje medio de PHA significativamente diferente a las demas. En términos paramétricos: \(H_1: \exists \alpha_i \neq 0\) para algún \(i \in \{1, 2, 3, 4\}\).

anova(modelo)

El p-valor es \(2.694 \times 10^{-6}\), un valor muy inferior a \(\alpha = 0.05\). Por tanto, rechazamos la hipótesis nula, lo que indica que existen diferencias estadisticamente significativas entre las diferentes fuentes de carbono.

Problema 2 (10%)

En el análisis de los datos de un experimento con un ANOVA de un factor correspondiente a un diseño balanceado, se dispone de la siguiente tabla incompleta:

Fuente.de.variación gl SS MS F
Grupo 5 11.8 20.6
Residuals 13.7
Total

Se pide:

1. Completar la tabla ANOVA

Para completar la tabla tenemos en cuenta las siguientes relaciones:

  • \(MS = SS / gl\), por lo que \(SS = MS \cdot gl\)

    • \(SS_{Grupo}=11.8\cdot 5=59\)

  • \(F = MS_{Grupo} / MS_{Residuals}\), por lo que \(MS_{Residuals} = MS_{Grupo} / F\)

    • \(MS_{Residuals} = 11.8 / 20.6= 0.57\)

  • \(gl_{Residuals} = SS_{Residuals} / MS_{Residuals}\)

    • \(gl_{Residuals} = 13.7 / 0.57=24\)

Por lo que tabla completa sería de la siguiente manera:

Fuente.de.variación gl SS MS F
Grupo 5 59.0 11.8 20.6
Residuals 24 13.7 0.57
Total 29 72.7

2. Si las medias de los grupos son: \(\bar{y}_1 = 4.6\), \(\bar{y}_2 = 5.3\), \(\bar{y}_3 = 4.5\), \(\bar{y}_4 = 6.8\), \(\bar{y}_5 = 7.1\), \(\bar{y}_6 = 8.3\), calcular el valor LSD y determinar qué comparaciones por parejas son significativas con un nivel de significación del 5%.

La formula de LSD es: \(LSD = t_{N-a, \epsilon/2} \sqrt{MSE \cdot \frac{2}{n}}\) y sabemos que:

  • \(n=5\)

  • \(a=6\)

  • \(N=30\)

  • \(\epsilon = 0.05\)

  • \(MSE=0.57\)

# Calculamos LSD
LSD=qt(0.975, 24)*sqrt(2*0.57/5)
print(LSD)
## [1] 0.9854981

Para determinar qué comparaciones por parejas son significativas, tenemos que calcular \(|\bar{y}_i - \bar{y}_j|\) de todos los pares.

\(|\bar{y}_{1\cdot} - \bar{y}_{2\cdot}| = |4.6 - 5.3| = 0.7\)

\(|\bar{y}_{1\cdot} - \bar{y}_{3\cdot}| = |4.6 - 4.5| = 0.1\)

\(|\bar{y}_{1\cdot} - \bar{y}_{4\cdot}| = |4.6 - 6.8| = 2.2\)

\(|\bar{y}_{1\cdot} - \bar{y}_{5\cdot}| = |4.6 - 7.1| = 2.5\)

\(|\bar{y}_{1\cdot} - \bar{y}_{6\cdot}| = |4.6 - 8.3| = 3.7\)

\(|\bar{y}_{2\cdot} - \bar{y}_{3\cdot}| = |5.3 - 4.5| = 0.8\)

\(|\bar{y}_{2\cdot} - \bar{y}_{4\cdot}| = |5.3 - 6.8| = 1.5\)

\(|\bar{y}_{2\cdot} - \bar{y}_{5\cdot}| = |5.3 - 7.1| = 1.8\)

\(|\bar{y}_{2\cdot} - \bar{y}_{6\cdot}| = |5.3 - 8.3| = 3.0\)

\(|\bar{y}_{3\cdot} - \bar{y}_{4\cdot}| = |4.5 - 6.8| = 2.3\)

\(|\bar{y}_{3\cdot} - \bar{y}_{5\cdot}| = |4.5 - 7.1| = 2.6\)

\(|\bar{y}_{3\cdot} - \bar{y}_{6\cdot}| = |4.5 - 8.3| = 3.8\)

\(|\bar{y}_{4\cdot} - \bar{y}_{5\cdot}| = |6.8 - 7.1| = 0.3\)

\(|\bar{y}_{4\cdot} - \bar{y}_{6\cdot}| = |6.8 - 8.3| = 1.5\)

\(|\bar{y}_{5\cdot} - \bar{y}_{6\cdot}| = |7.1 - 8.3| = 1.2\)

Las comparaciones por parejas que son significativas son las que son mayores a LSD, es decir, cuando son mayores a 0.98. En este caso, son las comporaciones 1-4, 1-5, 1-6, 2-4, 2-5, 2-6, 3-4, 3-5, 3-6, 4-6 y 5-6.

Problema 3 (10%)

1. Comentar brevemente (máximo 5 líneas) la respuesta proporcionada.

La respuesta de la IA explica que ambos métodos usan una misma fórmula, pero con distintos fines, ya que la regresión usa números (como la altura) para ver si hay tendencias y pendientes, y el ANOVA usa categorías (como colores) para comparar promedios y ver si los grupos son diferentes entre sí. En resumen, la diferencia principal está en el tipo de datos que utiliza para explicar el problema.

2. Comentar brevemente (máximo 5 líneas) la respuesta proporcionada.

En este caso, nos explica que si tenemos números y categorías a la vez, hay un modelo que se puede utilizar llamado ANCOVA. Este modelo permite observar como varía la tendencia según el grupo al que perteneces (ANOVA). En resumen, sirve para comparar grupos de manera mas justa, teniendo en cuenta que hay otros datos numéricos que influyen.
